## Dataset Description
The Titanic dataset contains information about passengers aboard the Titanic ship.
It includes features such as age, gender, passenger class, fare, and survival status.

## Machine Learning Problem
This project defines a supervised classification problem.
The objective is to predict whether a passenger survived based on demographic and travel-related features.

## Target Variable
- Survived (0 = Did not survive, 1 = Survived)
